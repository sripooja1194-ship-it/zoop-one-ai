# Zoop-One-AI

An AI-powered Career Assistant built using Streamlit, Groq LLM, and Whisper that helps users with resume analysis, interview preparation, career guidance, voice interactions, and website-based question answering.

## Features

* AI Resume Analysis
* Resume Improvement Suggestions
* Interview Question Generation
* Mock Interview Preparation
* Career Guidance Assistant
* Voice-to-Text Interaction using Whisper
* Website Chat and Content Analysis
* Conversational AI powered by LLMs

## Tech Stack

* Python
* Streamlit
* Groq LLM
* Whisper
* Generative AI
* Prompt Engineering
* Natural Language Processing (NLP)

## Project Highlights

Zoop-One-AI is an intelligent career development platform that leverages Large Language Models to assist job seekers with resume optimization, interview readiness, career planning, and interactive AI conversations through text and voice.
